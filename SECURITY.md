# Security Policy

## Supported Versions

The following versions of Smart Agri Assistant are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to the maintainers. All security vulnerabilities will be promptly addressed.

Please include the following information in your report:

* Type of issue (e.g., SQL injection, cross-site scripting, etc.)
* Full path of source file(s) related to the manifestation of the issue
* The location of the affected code (tag/branch/commit or direct URL)
* Any special configuration required to reproduce the issue
* Step-by-step instructions to reproduce the issue
* Proof-of-concept or exploit code (if possible)
* Impact of the issue, including how an attacker might exploit the issue

We prefer all communications to be in English.

You should receive a response within 48 hours. If for some reason you do not, please follow up via the same e-mail to ensure we received your original message.
